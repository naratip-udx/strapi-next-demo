# strapi-next-demo

Strapi CMS + Next.js Demo

```properties
# .env.local
NODE_ENV=development
STRAPI_BASE_URL=http://localhost:1337
STRAPI_API_URL=${STRAPI_BASE_URL}/api
```

frontend develop start: `cd frontend && pnpm dev`

backend develop start: `cd backend/app && npm run dev`

Reference:

- [Strapi - Quick Start Guide](https://docs.strapi.io/cms/quick-start)

- [Next.js - Getting Started](https://nextjs.org/docs/app/getting-started)
